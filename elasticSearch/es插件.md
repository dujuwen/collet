ElasticSearch-Head 是一个与 Elastic 集群（Cluster）相交互的 Web 前台。

ES-Head 的主要作用：
它展现 ES 集群的拓扑结构，并且可以通过它来进行索引（Index）和节点（Node）级别的操作
它提供一组针对集群的查询 API，并将结果以 json 和表格形式返回
它提供一些快捷菜单，用以展现集群的各种状态
